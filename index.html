Avalanche is actively integrating AI tooling to lower the barrier for developers and power next-generation dApps.

A strategic partnership with Aethir, a decentralized GPU cloud provider. Aethir's $100M Ecosystem Fund will support Avalanche's new infraBUIDL(AI) Program, providing essential GPU resources to AI projects building on Avalanche.

A new open-source tool that connects AI assistants (like Claude, or tools in an IDE) directly to the Avalanche network. This allows developers to use natural language commands ("tell") instead of complex code ("code") to perform blockchain operations, dramatically speeding up development.
